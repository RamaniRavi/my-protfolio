# Ravi Ramani – Portfolio

A personal portfolio website showcasing my work as a Software Engineer and Automation Engineer.

**Live sections:** Home · About · Experience · Education · Skills · Projects · Certificates · Contact

---

## Tech Stack

| Layer | Technologies |
|-------|-------------|
| Markup | HTML5 |
| Styling | CSS3, SCSS, Bootstrap 4 |
| Scripts | JavaScript, jQuery |
| Animations | AOS, Animate.css, Owl Carousel, Stellar.js |
| Icons | Font Awesome 6, Devicons, IcoMoon, Ionicons, Flaticon |

---
